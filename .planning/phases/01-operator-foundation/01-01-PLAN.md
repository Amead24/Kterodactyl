       - GameServerState type (string)
       - 6 state constants: Creating, Starting, Ready, Allocated, Shutdown, Error
       - ValidTransitions map defining allowed transitions:
         - Creating -> Starting, Error
         - Starting -> Ready, Error, Shutdown
         - Ready -> Allocated, Shutdown
         - Allocated -> Ready, Shutdown (Ready for de-allocation back to pool)
         - Shutdown -> (terminal, no transitions)
         - Error -> Shutdown (can only shutdown from error)
       - `IsValidTransition(from, to GameServerState) bool` function
       - `IsTerminal(state GameServerState) bool` function returning true for Shutdown
       - Status condition type constants: TypeReady = "Ready", TypeProgressing = "Progressing", TypeDegraded = "Degraded"
       - Finalizer constant: `FinalizerName = "game.kterodactyl.io/finalizer"`
