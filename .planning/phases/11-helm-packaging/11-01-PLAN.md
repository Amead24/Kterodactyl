**chart/templates/serviceaccount.yaml:**
Conditional on `.Values.serviceAccount.create`. Standard ServiceAccount with `{{ include "kterodactyl.fullname" . }}` name, Helm labels, optional annotations from values.

**chart/templates/deployment.yaml:**
Convert config/manager/manager.yaml to Helm template. Key differences from kustomize source:
- Use `{{ include "kterodactyl.fullname" . }}` for the Deployment name (NOT hardcoded)
- namespace: `{{ .Release.Namespace }}`
- Labels via `{{ include "kterodactyl.labels" . | nindent 4 }}`
- Selector matchLabels via `{{ include "kterodactyl.selectorLabels" . | nindent 6 }}`
- replicas: `{{ .Values.replicaCount }}`
- image: `"{{ .Values.image.repository }}:{{ .Values.image.tag | default .Chart.AppVersion }}"`
- imagePullPolicy: `{{ .Values.image.pullPolicy }}`
- command: ["/manager"]
- args: --leader-elect, --health-probe-bind-address=:8081, --metrics-bind-address=:8443, --api-bind-address=:8080, plus `{{ toYaml .Values.manager.extraArgs | nindent 8 }}` if set
- env: OPERATOR_NAMESPACE from fieldRef metadata.namespace (critical -- must always match deployment namespace)
- Three named ports: api (8080), metrics (8443), health (8081)
- livenessProbe: httpGet /healthz port 8081, initialDelaySeconds 15, periodSeconds 20
- readinessProbe: httpGet /readyz port 8081, initialDelaySeconds 5, periodSeconds 10
- resources from `.Values.manager.resources`
- Pod securityContext from `.Values.manager.podSecurityContext`
- Container securityContext from `.Values.manager.securityContext`
- serviceAccountName via `{{ include "kterodactyl.serviceAccountName" . }}`
- terminationGracePeriodSeconds: 10
- imagePullSecrets from `.Values.imagePullSecrets` if set
- nodeSelector, tolerations, affinity from values (with `{{- with }}` blocks)
- No volumeMounts or volumes needed (games dir is baked into the container image)

- name: `{{ include "kterodactyl.fullname" . }}-api`
- type: `{{ .Values.apiService.type }}`
- port: `{{ .Values.apiService.port }}` targeting containerPort 8080
- selector via `{{ include "kterodactyl.selectorLabels" . | nindent 4 }}`
- Helm labels
